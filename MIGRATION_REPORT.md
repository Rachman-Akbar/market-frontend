# React Feature-Driven Restructure

Folder `app/` Next-style sudah dihapus dari source dan diganti menjadi feature-driven structure di dalam `src/features`. File `page.jsx` dan `layout.jsx` nested sudah diganti menjadi nama halaman/komponen PascalCase.

Gunakan ZIP ini dengan mengganti folder `src` lama pada project Vite React kamu.
