Minimal perlu dikembangkan seperti ini:

ALTER TABLE vouchers
ADD COLUMN discount_target ENUM('order','shipping') NOT NULL DEFAULT 'order' AFTER name,
ADD COLUMN max_shipping_discount DECIMAL(10,2) NULL AFTER max_discount;

Lalu di tabel orders sebaiknya ada field seperti:

ALTER TABLE orders
ADD COLUMN shipping_fee DECIMAL(12,2) NOT NULL DEFAULT 0,
ADD COLUMN shipping_discount DECIMAL(12,2) NOT NULL DEFAULT 0,
ADD COLUMN voucher_discount DECIMAL(12,2) NOT NULL DEFAULT 0,
ADD COLUMN voucher_code VARCHAR(50) NULL;

Kalau discount_target = 'shipping', maka kalkulasinya diarahkan ke ongkir, bukan subtotal barang. Kalau belum ada kalkulasi ongkir di order/checkout, voucher gratis ongkir belum bisa bekerja benar meskipun data vouchernya sudah ada.
